## 🌟 Features

### 🎯 Core Gameplay
- **10 Hand-Crafted Stages** — Each ends with a unique boss with its own attack pattern
- **Boss Variety** — From Recon Drone to The Singularity, every boss feels different
- **Procedural Audio** — All sound effects and ambient music synthesized in code
- **Endless Mode** — Unlocked after clearing all 10 stages; infinite scaling difficulty
- **Portrait Orientation** — Designed for vertical play, perfect for modern monitors and mobile windows

### ⚡ Power-Ups
| Power-Up | Effect | Rarity |
|----------|--------|--------|
| 🔵 **Triple Shot** | 3-way spread for 10 seconds | Medium |
| 🟣 **Quintuple** | 5-way spread for 9 seconds | Low |
| 🟠 **Missile Overcharge** | Unlimited missiles for 6 seconds | Medium |
| 🔷 **Shield** | Damage immunity for 6 seconds | Medium |
| ⚪ **Extra Life** | +1 heart (or +300 score if at max) | Low |

### 🎨 Visual & Audio
- **Parallax Starfield** — Layered stars + drifting nebula clouds
- **Glow Effects** — Neon-style soft glow on ships, bullets, and power-ups
- **Synthesized Soundtrack** — Dark sci-fi ambient pad (no external audio files)
- **Procedural SFX** — Lasers, missiles, explosions, power-ups, and boss alerts
- **Resizable Window** — Maintains portrait aspect ratio with letterboxing

### 🎮 Controls
| Action | Keys / Mouse |
|--------|--------------|
| Move Ship | WASD / Arrow keys or **Mouse** |
| Fire Blaster | Hold SPACE or **Hold Left Click** |
| Fire Missile | Tap LSHIFT or **Right Click** (5s cooldown) |
| Pause | P, ESC, or **click pause icon** (top-left) |
| Toggle Cursor | CTRL or ALT |
| Restart | R (on Game Over / Victory) |
| Quick Mute | M |
| Quit | ESC (from main menu) |

## 🚀 Quick Start

### 1. Install Dependencies

```bash
pip install pygame numpy
```

### 2. Run the Game

```bash
python void_breaker.py
```

### 3. Play!

- Navigate the main menu
- Choose **NEW GAME** to start from Stage 1
- Or **CONTINUE** to pick up from your best progress
- Unlock **STAGE SELECT** and **ENDLESS MODE** by beating all 10 stages!
